In order to prove that log( n 70 ) = O (log( n 1 / 2 )), we need to show that there exists constants c and n 0 such that log( n 70 ) ≤ c log( n 1 / 2 ) for all n > n 0 . Let c = 140. Then log( n 70 ) = 70 log( n ) = 140 * 1 2 log( n ) = 140 log( n 1 / 2 ) Thus, when we choose c = 140, we ﬁnd that for all n , log( n 70 ) = c log( n 1 / 2 ). This means that we can choose whatever value of n 0 that we want, and these two functions will satisfy the deﬁnition of big O. Part (b) False. Recall that 2 n = Θ( e n ) if both 2 n = O ( e n ) and 2 n = Ω( e n ). Thus, in order to prove that 2 n is not Θ( e n ), we need to show at at least one of these conditions does not hold. It happens that 2 n is in fact O ( e n ) (though we will not prove that here). So let’s prove by contradiction that 2 n is not Ω( e n ). Suppose 2 n = Ω( e n ). That is, suppose there exists some c and n 0 such that, for all n > n 0 , 2 n ≥ ce n . Rearranging the terms in the inequality, we ﬁnd that c ≤ ( 2 e ) n for all n > n 0 . But clearly this is impossible, because when n is suﬃciently large ( 2 e ) n is less than c . Thus we have a contradiction! 2 n cannot be Ω( e n ). 1
